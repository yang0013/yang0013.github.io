如何用Github Page和hexo建立个人博客

By Yang

发表于5月 19 2014

文章目录
  1. github上新建repository
  2. 在Dodaddy购买域名
  3. 在DNSpod设置DNS
  4. 本地安装配置git，node，hexo

Github Pages：免费的静态站点，特点：搭建简单，支持自制页面和Jekyll，可以绑定你的域名

登录Github，新建repository，github提供两种方式建立Pages，这里采用 “User & Organiztion Pages”，注意名称必须是 github_username.github.io， 官网的说明：

>
User & Organization Pages live in a special repository dedicated to GitHub Pages files. You will need to name this repository with the account name, e.g. atmos/atmos.github.io

  • You must use the username/username.github.io naming scheme.
  • Content from the master branch will be used to build and publish your GitHub Pages site.

因此我的项目名称为：yang0013.github.io

在Dodaddy购买域名
网上可以搜到优惠码，但使用优惠码的无法使用支付宝进行支付。普通域名$12.99，使用优惠码$8.99，便宜4美元。
域名购买成功后，登录后，点击域名，在Domain Settings 中添加 Nameservers， nameserver为dnspod提供的
F1G1NS1.DNSPOD.NET
F1G1NS2.DNSPOD.NET

注册DNSpod账号，并设置DNS
添加两个A记录和CNAME记录

主机记录 记录类型 线路类型 记录值 MX优先 TTL
@ A 默认 192.30.252.153 - 600
@ A 默认 192.30.252.154 - 600
www CNAME 默认 yang0013.github.io. - 600

A记录地址可以在
Github Page查看：
>
Apex domains
The simplest way to configure an apex domain for your GitHub Pages site is to create A records that resolve to the following IP addresses:
    192.30.252.153
   192.30.252.154

CNAME记录值为Github的repository地址


安装Git：
添加ssh信任，生成公钥文件： 

$ ssh-keygen -t rsa -C "邮件地址@youremail.com"  
$ cd ~/.ssh
$ cat id_rsa.pub

登录github，菜单Account Settings->SSH Public keys->add another public key,将id_rsa.pub的内容拷贝到文本框中。
测试 

$ssh -T git@github.com

测试用户信息 

$ git config --global user.name "yang0013"        // 用户名
$ git config --global user.email  "github注册邮箱"  // 填写自己的注册邮箱

安装hexo
安装node之后， 

npm install -g hexo

初始化，新建一个目录，进入目录执行命令 

hexo init    
hexo g  
hexo s

克隆主题

git clone https://github.com/wuchong/jacman.git themes/jacman

修改hexo目录下的_config.yml，将主题修改为jacman, 并设置github提交地址

theme: jacman
....
deploy:
type: github
## repo填写自己的github repository地址
repo: https://github.com/yang0013/yang0013.github.io.git  
branch: master

更新主题 

cd theme/jacman
git pull

新建CNAME文件， 

vi source/CNAME

文件内容只有一行： yangact.com，此域名为在godaddy申请到的域名。保存文件后，
提交更新至github代码仓库中 

hexo d -g

此时，设置完成。可以访问自己的主页了。根据需要修改themes/jacman/_config.yml。
